# KWin

KWin is best configured through the GUI rather than a drop-in config file.

1. Open **System Settings → Window Management → Window Rules → Add New**
2. Set **Window class (regex)** to `goose-.*`
3. Adjust **Keep above** or other grouping behavior to suit your workflow
4. Plasma 6 no longer has the old built-in window tabbing from Plasma 5, so the closest equivalents are TabBox workflows or third-party scripts
